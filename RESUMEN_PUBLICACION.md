```
├─ Visibilidad: Público
└─ Create repository
```

### 2️⃣ Subir código (1 min)
```bash
# En tu terminal, en la carpeta del proyecto:
git remote add origin https://github.com/Jonathan150692/totem-flow.git
git push -u origin main

# Te pedirá contraseña → COPIA TU TOKEN DE:
# https://github.com/settings/tokens → Generate new token → Scope "repo"
```

### 3️⃣ Publicar en Vercel (2 min)
```
https://vercel.com/sign-up
├─ Sign up with GitHub
├─ Add New → Project
├─ Import totem-flow
├─ Deploy (sin configurar nada)
└─ URL en ~60 segundos: https://totem-flow.vercel.app
```
