🎯 QUÉ HACER AHORA
─────────────────────────────────────────────────────────────────────────────────

OPCIÓN A: Si quieres ver el prototipo corriendo LOCALMENTE (en tu máquina)
──────────────────────────────────────────────────────────────────────────────

1. Instala Node.js desde https://nodejs.org (la versión LTS)

2. En la terminal, navega a esta carpeta:
   cd /ruta/a/esta/carpeta

3. Instala las dependencias:
   npm install

4. Corre en desarrollo (abre http://localhost:3000):
   npm run dev

5. Verás el Dashboard, Kanban y Checklist dinámico en vivo

OPCIÓN B: Si quieres publicar ONLINE (para que todos tus colaboradores vean)
───────────────────────────────────────────────────────────────────────────

Sigue la GUÍA COMPLETA en el archivo: PUBLICAR.md

TL;DR:
   1. Crea cuenta GitHub (https://github.com/signup)
   2. Crea repositorio nuevo
   3. Sube este código: git init → git add . → git commit → git push
   4. Ve a https://vercel.com/sign-up → conecta con GitHub
   5. Importa el repo → Vercel auto-deploya
   6. Tu URL estará lista en ~60 segundos: https://totem-flow.vercel.app

🎨 VISTA PREVIA DEL PROTOTIPO
─────────────────────────────────────────────────────────────────────────────────

El código corre estas 3 vistas principales:

1. DASHBOARD
   • KPIs: Proyectos activos, cumplimiento SLA, vencidos, valor en pipeline
   • Gráfico de distribución por fase
   • Proyectos próximos a vencer (alertas)
   • Tabla con todos los proyectos

2. KANBAN (Vista por fase)
   • 5 columnas: Revisión → Checklist → Reunión → Fase Ninja → Entregado
   • Tarjetas arrastrables con SLA, soluciones, días abiertos, preguntas

3. DETALLE DE PROYECTO
   • Header con info: cliente, ejecutivo, localidades, valor
   • 6 tabs: Resumen, Checklist, Preguntas, Reunión, Documentos, Bitácora
   • Checklist dinámico funcional:
     - Secciones por solución (CCTV, Control de Acceso, Aprobación Financiera)
     - Ítems Completado/Pendiente/No aplica
     - Barra de progreso en tiempo real
     - Datos de ejemplo reales de TOTEM

Datos de ejemplo incluyen 6 clientes reales:
   • Country Club Quito
   • CNT EP Matriz
   • Grupo Almar (2 proyectos)
   • Promarisco
   • SENAE Guayaquil
